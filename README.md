# silicon-logic-site

Source for [siliconlogic.dev](https://siliconlogic.dev) — the public-facing publication for Silicon Logic.

The editorial substrate (signed benchmark runs, methodology, canonical article Markdown) lives at [Vargix/silicon-logic](https://github.com/Vargix/silicon-logic). This repo is the renderer that turns that substrate into a publication readers can read.

Pre-launch. First article in progress.
